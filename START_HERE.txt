💡 HOW IT WORKS:

All three methods bypass the network firewall by:

1. YOU navigate using your logged-in browser
2. JavaScript extracts data from the page you're viewing
3. Data is saved locally in your browser
4. You download the collected data as JSON

Since we're using YOUR browser (not automation tools), the network
firewall doesn't block it. Your browser already has access!

═══════════════════════════════════════════════════════════════════════════

📊 WHAT DATA GETS COLLECTED:

For each course page, we extract:

✓ Course title
✓ Entry requirements (all cluster subjects and grades)
✓ Subject requirements (all subjects and minimum grades)
✓ Available programmes:
  - Institution name
  - County
  - Programme type
  - Programme code
  - Programme name
  - KCSE 2025 cutoff points
  - KCSE 2024 cutoff points
  - KCSE 2023 cutoff points
  - Cluster weights

All data is extracted automatically from the tables on each page.

🔧 IF SOMETHING GOES WRONG:

Issue: Bookmark doesn't work
Fix: Make sure you copied the ENTIRE bookmarklet code, including
     "javascript:" at the beginning

Issue: No popup appears
Fix: Allow popups for the KUCCPS website in your browser settings

Issue: Want to start over
Fix: Open Console (F12) and run: localStorage.removeItem('kuccps_scraped_courses')

Issue: Lost progress
Fix: Don't worry! Data is saved in browser. Just click bookmark again
     on any page to see your count, then continue
